# 一、题目描述

## **问题概括**

给定一个字符串 `s`，它由 ASCII 字符集中的可打印字符组成。请你编写一个函数，将这个字符串中的**所有大写字母**转换为对应的小写字母，而其他字符保持不变，并返回转换后的新字符串。

## **示例说明**

- **示例1**：输入 `s = "Hello"`，输出 `"hello"`。
- **示例2**：输入 `s = "here"`（全为小写），输出仍为 `"here"`。
- **示例3**：输入 `s = "LOVELY"`（全为大写），输出为 `"lovely"`。

## **数据范围**

- 字符串的长度在 `1` 到 `100` 之间。
- 字符串仅包含 ASCII 可打印字符。



# 二、解题思路\解题过程

这也是一道简单的题目，可以采用循环将字符串中符合条件的字符进行转换，也可以直接使用内置函数，代码如下：
```java
class Solution {
    public String toLowerCase(String s) {
        return s.toLowerCase();
    }
}
```

## Python代码

```python
class Solution:
    def toLowerCase(self, s: str) -> str:
        return s.lower()
```



## C++代码

```C++
class Solution {
public:
    string toLowerCase(string s) {
        for (char &str : s) {
            str = tolower(static_cast<unsigned char>(str));
        }
        return s;
    }
};
```
