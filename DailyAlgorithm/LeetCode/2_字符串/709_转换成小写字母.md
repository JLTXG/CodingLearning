[709. 转换成小写字母 - 力扣（LeetCode）](https://leetcode.cn/problems/to-lower-case/description/?envType=study-plan-v2&envId=primers-list)

>标签：字符串

# 一、题目描述

给你一个字符串 `s` ，将该字符串中的大写字母转换成相同的小写字母，返回新的字符串。

 

**示例 1：**

```
输入：s = "Hello"
输出："hello"
```

**示例 2：**

```
输入：s = "here"
输出："here"
```

**示例 3：**

```
输入：s = "LOVELY"
输出："lovely"
```

 

**提示：**

- `1 <= s.length <= 100`
- `s` 由 ASCII 字符集中的可打印字符组成



# 二、解题思路\解题过程

这也是一道简单的题目，可以采用循环将字符串中符合条件的字符进行转换，也可以直接使用内置函数，代码如下：
```java
class Solution {
    public String toLowerCase(String s) {
        return s.toLowerCase();
    }
}
```



# 三、代码演示

## Python代码

```python
class Solution:
    def toLowerCase(self, s: str) -> str:
        return s.lower()
```



## C++代码

```C++
class Solution {
public:
    string toLowerCase(string s) {
        for (char &str : s) {
            str = tolower(static_cast<unsigned char>(str));
        }
        return s;
    }
};
```
