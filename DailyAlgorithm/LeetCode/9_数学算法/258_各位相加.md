# 一、题目描述

## **问题概括**

给定一个非负整数 `num`，反复计算其十进制表示下各位数字之和，直到结果变为一个一位数（即 0 到 9 之间）。返回最终得到的一位数。

## **数据范围**

- `0 <= num <= 2^31 - 1`



# 二、解题思路\解题过程

这是一道简单的题目，两层循环就可以搞定，话不多说，代码如下：
```java
class Solution {
    public int addDigits(int num) {
        while(num >= 10){
            int flag = 0;
            while(num != 0){
                flag += num % 10;
                num /= 10;
            }
            num = flag;
        }
        return num;
    }
}
```
