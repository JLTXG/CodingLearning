# 一、题目描述

## **问题概括**

给定一个表示摄氏温度的浮点数 `celsius`，按照以下公式计算对应的开氏温度（Kelvin）和华氏温度（Fahrenheit）：

- 开氏温度 = 摄氏温度 + 273.15
- 华氏温度 = 摄氏温度 × 1.80 + 32.00

返回一个包含两个浮点数的数组 `[kelvin, fahrenheit]`。计算结果允许与实际值有不超过 `10^-5` 的误差。

## **示例说明**

- **示例1**：输入 `celsius = 36.50`，计算得到开氏温度 = 36.50 + 273.15 = 309.65，华氏温度 = 36.50 × 1.80 + 32.00 = 97.70，输出 `[309.65000, 97.70000]`。
- **示例2**：输入 `celsius = 122.11`，计算得到开氏温度 = 122.11 + 273.15 = 395.26，华氏温度 = 122.11 × 1.80 + 32.00 = 251.798，输出 `[395.26000, 251.79800]`。

## **数据范围**

- `0 <= celsius <= 1000`



# 二、解题思路\解题过程

这也是一道非常简单的题目，就是普通的数学题，按照题目要求来写就好了，话不多说，代码如下：

```cpp
class Solution {
public:
    vector<double> convertTemperature(double celsius) {
        double kelvin = celsius + 273.15, fahrenheit = celsius * 1.80 + 32.00;
        vector<double> values{kelvin, fahrenheit};
        return values;
    }
};
```
