# 一、题目描述

## **问题概括**

给定一个正整数 `n`，计算并返回 `2` 和 `n` 的最小公倍数（LCM）。最小公倍数是指能同时被 `2` 和 `n` 整除的最小正整数。

## **示例说明**

- **示例1**：`n = 5`，2 和 5 的最小公倍数是 10。
- **示例2**：`n = 6`，2 和 6 的最小公倍数是 6。

## **数据范围**

- `1 <= n <= 150`



# 二、解题思路\解题过程

这是一道简单题目，就是单纯的数学题目，求得是两个数的最小偶倍数，只不过与求两个数的最小公倍数的区别在于，这道题已经将其中一个数`2`明牌了，通过<mark>辗转相除法</mark>——（又称**欧几里得算法**，Euclidean Algorithm）是一种用于**求两个非负整数的最大公约数**（GCD）的经典高效算法——来求出两个数的最大公约数，然后通过`乘数1 / 最大公约数 * 乘数2`得到两个数的最小偶倍数！代码如下：

```java
class Solution {
    public int smallestEvenMultiple(int n) {
        int a = 2, b = n;
        while(b != 0){
            int tmp = b;
            b = a % b;
            a = tmp;
        }
        return n / a * 2;
    }
}
```
