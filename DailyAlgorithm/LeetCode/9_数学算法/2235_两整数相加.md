# 一、题目描述

## **问题概括**

给定两个整数 `num1` 和 `num2`，它们的取值范围均在 `-100` 到 `100` 之间。你需要计算并返回这两个整数的和。

## **示例说明**

- **示例1**：`num1 = 12`, `num2 = 5`，和为 `17`。
- **示例2**：`num1 = -10`, `num2 = 4`，和为 `-6`。

## **数据范围**

- `-100 <= num1, num2 <= 100`



# 二、解题思路\解题过程

这是一道非常简单的题目，就是将题目给的两个整数相加，然后在返回即可，而且，也不用考虑==溢出==的问题，因为题目给出的整数的范围为`-100 <= num1, num2 <= 100`，代码如下：

```java
class Solution {
    public int sum(int num1, int num2) {
        return num1 + num2;
    }
}
```
