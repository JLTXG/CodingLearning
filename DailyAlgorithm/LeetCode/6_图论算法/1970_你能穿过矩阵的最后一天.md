# 一、题目描述

## **问题概括**

你有一个 `row` 行 `col` 列的网格，其中每个格子要么是陆地（用 `0` 表示），要么是水域（用 `1` 表示）。初始时（第 0 天），整个网格都是陆地。

接下来，每天会有一个指定的陆地格子变成水域，变化顺序由一个二维数组 `cells` 给出。具体来说，在第 `i` 天（`i` 从 1 开始计数），位于第 `ri` 行、第 `ci` 列（行列下标均从 1 开始）的陆地格子会变成水域。

你的任务是，找出**最后一天**（即最大的天数），使得在这一天结束时，仍然存在一条从网格**最上面一行**（第一行）到**最下面一行**（最后一行）的路径，该路径只能经过陆地格子（即值为 `0` 的格子）。你可以从第一行的任意陆地格子出发，到达最后一行的任意陆地格子，且只能向上下左右四个方向移动。

如果从某一天开始，不再存在这样的路径，那么前一天就是满足条件的最后一天。你需要返回这个天数。



## **示例核心说明**

- **示例1**：一个 `2x2` 网格，变化顺序为 `[[1,1],[2,1],[1,2],[2,2]]`。在第 2 天结束时，仍然可以从第一行走到第二行；但第 3 天之后就不再可能。因此最后一天是 `2`。

- **示例3**：一个 `3x3` 网格，变化顺序较长，最后一天是 `3`。

输出：3
解释：上图描述了矩阵从第 0 天开始是如何变化的。

## **数据范围**

- `2 <= row, col <= 2 * 10^4`
- 网格总格子数满足 `4 <= row * col <= 2 * 10^4`
- `cells` 的长度等于 `row * col`，即所有格子最终都会变成水域
- `cells` 中的每个坐标都是唯一的
