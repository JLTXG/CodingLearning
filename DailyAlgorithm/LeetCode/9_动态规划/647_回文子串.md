# 一、题目描述

给你一个字符串 `s` ，请你统计并返回这个字符串中 **回文子串** 的数目。

**回文字符串** 是正着读和倒过来读一样的字符串。

**子字符串** 是字符串中的由连续字符组成的一个序列。

 

**示例 1：**

```
输入：s = "abc"
输出：3
解释：三个回文子串: "a", "b", "c"
```

**示例 2：**

```
输入：s = "aaa"
输出：6
解释：6个回文子串: "a", "a", "a", "aa", "aa", "aaa"
```

 

**提示：**

- `1 <= s.length <= 1000`
- `s` 由小写英文字母组成

# 三、代码演示

## Java代码

```java
class Solution {
    public int countSubstrings(String s) {
        int n = s.length(), result = 0;
        boolean[][] dp = new boolean[n][n];
        for(int i = n - 1; i >= 0; i--){
            for(int j = i; j < n; j++){
                if(s.charAt(i) == s.charAt(j)){
                    if(j - i <= 1){
                        dp[i][j] = true;
                        result++;
                    } else if (dp[i + 1][j - 1]){
                        dp[i][j] = true;
                        result++;
                    }
                }
            }
        }
        return result;
    }
}
```



## Python代码

```python
class Solution:
    def countSubstrings(self, s: str) -> int:
        result, s_len = 0, len(s)
        dp = [[0] * s_len for _ in range(s_len)]
        for i in range(s_len - 1, -1, -1):
            for j in range(i, s_len, 1):
                if s[i] == s[j] :
                    if j - i <= 1 :
                        dp[i][j] = 1
                        result += 1
                    elif dp[i + 1][j - 1] == 1 :
                        dp[i][j] = 1
                        result += 1
        return result
```

        for(int i = n - 1; i >= 0; i--){
            for(int j = i; j < n; j++){
                if(s[i] == s[j]){
                    if(j - i <= 1){
                        dp[i][j] = true;
                        result++;
                    } else if (dp[i + 1][j - 1]){
                        dp[i][j] = true;
                        result++;
                    }
                }
            }
        }
        return result;
    }
};
