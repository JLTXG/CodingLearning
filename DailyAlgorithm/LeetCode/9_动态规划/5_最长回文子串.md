# 一、题目描述

## **问题概括**

给定一个字符串 `s`，你需要找出并返回 `s` 中最长的 **回文子串**。回文子串是指正着读和反着读都一样的连续字符串片段。

## **示例说明**

- **示例1**：输入 `s = "babad"`，最长回文子串可以是 `"bab"` 或 `"aba"`，因此输出任意一个即可（通常输出第一个找到的）。
- **示例2**：输入 `s = "cbbd"`，最长回文子串是 `"bb"`。

## **数据范围**

- 字符串长度在 `1` 到 `1000` 之间。
- 字符串仅由数字和英文字母组成。

# 三、代码演示

## Java代码

```java
class Solution {
    public String longestPalindrome(String s) {
        int n = s.length(), flag = 0, s_i = 0, s_j = 0;
        boolean[][] dp = new boolean[n][n];
        for(int i = n - 1; i >= 0; i--){
            for(int j = i; j < n; j++){
                if(s.charAt(i) == s.charAt(j)){
                    if(j - i <= 1){
                        dp[i][j] = true;
                    } else if (dp[i + 1][j - 1]){
                        dp[i][j] = true;
                    }
                }
                if(dp[i][j] == true && (j - i + 1) > flag){
                    flag = flag > (j - i + 1) ? flag : (j - i + 1);
                    s_i = i;
                    s_j = j;
                }
            }
        }
        String newStr = "";
        for(int i = s_i; i <= s_j; i++){
            newStr += s.charAt(i);
        }
        return newStr;
    }
}
```

class Solution:
    def longestPalindrome(self, s: str) -> str:
        n, flag, s_i, s_j = len(s), 0, 0, 0
        dp = [[0] * n for _ in range(n)]
        for i in range(n - 1, -1, -1):
            for j in range(i, n, 1):
                if s[i] == s[j] :
                    if j - i <= 1 :
                        dp[i][j] = 1
                    elif dp[i + 1][j - 1] == 1 :
                        dp[i][j] = 1
                if dp[i][j] == 1 and (j - i + 1) > flag :
                    flag = j - i + 1
                    s_i = i
                    s_j = j
        new_str = ""
        for i in range(s_i, s_j + 1):
            new_str += s[i]
        return new_str
