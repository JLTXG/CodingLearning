# 一、题目描述

## **问题概括**

给定两个仅包含小写英文字母的字符串 `text1` 和 `text2`。**子序列** 的定义是：从原字符串中删除零个或多个字符（不改变剩余字符的相对顺序）后得到的新字符串。例如，`"ace"` 是 `"abcde"` 的一个子序列。

你需要找到两个字符串的 **最长公共子序列**，即同时存在于两个字符串中的最长子序列，并返回其长度。如果两个字符串没有公共子序列，则返回 `0`。

## **示例说明**

- **示例1**：`text1 = “abcde”`, `text2 = “ace”`，最长公共子序列为 `“ace”`，长度为 `3`。
- **示例2**：`text1 = “abc”`, `text2 = “abc”`，最长公共子序列为 `“abc”`，长度为 `3`。
- **示例3**：`text1 = “abc”`, `text2 = “def”`，没有公共子序列，返回 `0`。

## **数据范围**

- `1 <= text1.length, text2.length <= 1000`
- 两个字符串仅包含小写英文字母。

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        n, m = len(text1), len(text2)
        dp = [[0] * (m + 1) for _ in range(n + 1)]
        for i in range(1, n + 1):
            for j in range(1, m + 1):
                if text1[i - 1] == text2[j - 1] :
                    dp[i][j] = dp[i - 1][j - 1] + 1
                else :
                    dp[i][j] = dp[i - 1][j] if dp[i - 1][j] > dp[i][j - 1] else dp[i][j - 1]
        return dp[n][m]
```
