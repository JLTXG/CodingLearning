# 一、题目描述

## **问题概括**

给定一个由小写英文字母组成的字符串 `s`，你需要统计并返回其中所有 **回文子串** 的数量。回文子串是指正读和反读都相同的连续字符序列。注意，即使多个子串的内容相同，只要它们起始或结束的位置不同，就被视为不同的子串。

## **示例说明**

- **示例1**：输入 `s = "abc"`，共有三个回文子串：`"a"`、`"b"`、`"c"`，因此输出 `3`。
- **示例2**：输入 `s = "aaa"`，回文子串包括：三个单独的 `"a"`、两个 `"aa"`（分别对应下标 `[0,1]` 和 `[1,2]`）、以及一个 `"aaa"`，共计 `6` 个。

## **数据范围**

- 字符串长度在 `1` 到 `1000` 之间。
- 字符串仅由小写英文字母组成。

# 三、代码演示

## Java代码

```java
class Solution {
    public int countSubstrings(String s) {
        int n = s.length(), result = 0;
        boolean[][] dp = new boolean[n][n];
        for(int i = n - 1; i >= 0; i--){
            for(int j = i; j < n; j++){
                if(s.charAt(i) == s.charAt(j)){
                    if(j - i <= 1){
                        dp[i][j] = true;
                        result++;
                    } else if (dp[i + 1][j - 1]){
                        dp[i][j] = true;
                        result++;
                    }
                }
            }
        }
        return result;
    }
}
```



## Python代码

```python
class Solution:
    def countSubstrings(self, s: str) -> int:
        result, s_len = 0, len(s)
        dp = [[0] * s_len for _ in range(s_len)]
        for i in range(s_len - 1, -1, -1):
            for j in range(i, s_len, 1):
                if s[i] == s[j] :
                    if j - i <= 1 :
                        dp[i][j] = 1
                        result += 1
                    elif dp[i + 1][j - 1] == 1 :
                        dp[i][j] = 1
                        result += 1
        return result
```
