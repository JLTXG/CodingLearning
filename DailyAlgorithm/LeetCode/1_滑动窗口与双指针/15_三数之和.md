# 一、题目描述

## **问题概括**

给定一个整数数组 `nums`，请找出所有和为 0 的三元组 `[a, b, c]`（其中 `a`、`b`、`c` 是数组中三个不同位置的元素）。返回的结果中不应包含重复的三元组（即值相同的三元组只算一次，不考虑顺序）。

## **示例说明**

- **示例1**：`nums = [-1,0,1,2,-1,-4]`，和为 0 的三元组有 `[-1,-1,2]` 和 `[-1,0,1]`。
- **示例2**：`nums = [0,1,1]`，没有和为 0 的三元组，返回空列表 `[]`。
- **示例3**：`nums = [0,0,0]`，唯一的三元组 `[0,0,0]` 和为 0，返回 `[[0,0,0]]`。

## **数据范围**

- `3 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`



# 二、解题思路\解题过程

这道题目可以采用双指针的方法来求解，我们需要先创建对应的返回值的答案`result`变量，然后对题目提供的整个数组进行升序排序，然后，接下来有个小巧思，如果升序后的数组的第一个值——`nums[0]`——是大于0的，那我们就可以直接返回空的`result`了，因为如果第一个数组值（整个数组都升序了）都大于0了，那么后面无论哪三个数组值相加，那都是大于0的；然后，接下来，就是循环整个数组，如果其中遇到了重复的值，那么就跳过本次循环，然后我们接下来就需要定义两个指针——`left = i + 1, right = nums.size() - 1`，然后接下来就去遍历剩下的数组值，看是否有满足`nums[i] + nums[left] + nums[right] == 0`，如果大于0了，那么就`right--`，如果小于0，那么就`left++`，如果刚好等于0，那么就把这三个值`[nums[i], nums[left], nums[right]]`加入`result`中，并使`right++, left--`，我们还需要判断`nums[right]`是否等于`nums[right - 1]`，以及`nums[left]`是否等于`nums[left + 1]`，如果相等了，那么就`right--, left++`，直到不相等为止！

最后等循环完最外层的循环之后，就可以返回答案了！代码如下！

# 三、代码演示

## Java代码

```Java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> result = new ArrayList<>();
        Arrays.sort(nums);
        if (nums[0] > 0) {
            return result;
        }
        int n = nums.length;
        for (int i = 0; i < n; i++) {
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }
            int left = i + 1, right = n - 1;
            while (left < right) {
                int tmp = nums[i] + nums[left] + nums[right];
                if (tmp > 0) {
                    right--;
                } else if (tmp < 0) {
                    left++;
                } else {
                    result.add(Arrays.asList(nums[i], nums[left], nums[right]));
                    while (left < right && nums[right] == nums[right - 1]) {
                        right--;
                    }
                    while (left < right && nums[left] == nums[left + 1]) {
                        left++;
                    }
                    right--;
                    left++;
                }
            }
        }
        return result;
    }
}
```

## Python代码

```Python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        n, result = len(nums), []
        nums.sort()
        if nums[0] > 0:
            return result
        for i in range(n):
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            left, right = i + 1, n - 1
            while left < right:
                tmp = nums[i] + nums[left] + nums[right]
                if tmp > 0:
                    right -= 1
                elif tmp < 0:
                    left += 1
                else:
                    result.append([nums[i], nums[left], nums[right]])
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1
                    right -= 1
                    left += 1
        return result
```
