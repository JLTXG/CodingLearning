# 一、题目描述

## **问题概括**

给定一个整数数组 `cardPoints`，表示一排卡牌的点数。你每次可以从这一排的 **最左端** 或 **最右端** 取走一张卡牌，一共需要取 **恰好** `k` 张牌。你的最终得分为所取卡牌的点数之和。请计算你能够获得的最大点数。

## **示例说明**

- **示例1**：`cardPoints = [1,2,3,4,5,6,1]`, `k = 3`。最优方案是取最右边的三张牌（点数分别为 1、6、5），总和为 12。
- **示例2**：`cardPoints = [2,2,2]`, `k = 2`。无论取哪两张牌，总和均为 4。
- **示例3**：`cardPoints = [9,7,7,9,7,7,9]`, `k = 7`。必须取走全部卡牌，总和为 55。
- **示例4**：`cardPoints = [1,1000,1]`, `k = 1`。只能取两端点数为 1 的卡牌，最大点数为 1。
- **示例5**：`cardPoints = [1,79,80,1,1,1,200,1]`, `k = 3`。一种最优方案是取最左边的 1、最右边的 1 以及次右边的 200，总和为 202。

## **数据范围**

- `1 <= cardPoints.length <= 10^5`
- `1 <= cardPoints[i] <= 10^4`
- `1 <= k <= cardPoints.length`



# 二、解题思路\解题过程

这也是一道“滑动窗口”类的题目，这道题目可以用两种不同的思维方式来求解——逆向思维+正向思维，具体内容如下所示：
## 逆向思维

题目要从拿走`k`张卡牌，总共有`n`张卡牌，那么就只剩下`n - k`张卡牌了，另外，这个`k`张卡牌只能从首或者从尾开始拿，在拿够`k`张卡牌之后，剩下的`n - k`也必然是连续的卡牌，那么接下来，整个题目就转换成了`求数组中连续 n - k 个连续子数组和的最小值`。

在转换题目意思后，就可以采用灵神总结的“滑动窗口”类的经典解题方法——“**入——更新——出**”——就可以求出最终的答案了，代码如下：

```cpp
class Solution {
public:
    int maxScore(vector<int>& cardPoints, int k) {
        int windowSum = 0, n = cardPoints.size();
        int m = n - k;
        // 初始化第一个窗口
        for (int i = 0; i < m; i++) {
            windowSum += cardPoints[i];
        }

        int sum = windowSum, result = windowSum;
        for (int i = m; i < n; i++) {
            // 计算数组的总和
            sum += cardPoints[i];
            // 入 + 出
            windowSum += cardPoints[i] - cardPoints[i - m];
            // 更新答案
            result = min(result, windowSum);
        }
        
        return sum - result;
    }
};
```

## 正向思维

我们也可以采用正向思维来进行求解，正向思维也很简单，题目要求的卡牌只能从“首”或者“尾”进行拿取：
- 可以只拿首部的`k`张
- 可以只拿尾部的`k`张
- 可以先从首部开始拿，然后再从尾部拿
- 可以先从尾部开始拿，然后再从首部拿

所以，我们简单点，先从首部开始，求出第一个窗口内的总和，然后减去首部窗口内的对应数组值，然后在加上尾部的值，逐步更新，求出题目要求的`k`张卡牌值的最大总和！

代码如下：

```java
class Solution {
    public int maxScore(int[] cardPoints, int k) {
        int windowSum = 0, n = cardPoints.length;
        // 初始化第一个窗口
        for (int i = 0; i < k; i++) {
            windowSum += cardPoints[i];
        }

        int result = windowSum;
        for (int i = 1; i <= k; i++) {
            // 入 + 出
            windowSum += cardPoints[n - i] - cardPoints[k - i];
            // 更新答案
            result = Math.max(result, windowSum);
        }
        
        return result;
    }
}
```

# 逆向思维
class Solution:
    def maxScore(self, cardPoints: List[int], k: int) -> int:
        window_sum, n = 0, len(cardPoints)
        m = n - k
        # 初始化第一个窗口
        for i in range(m):
            window_sum += cardPoints[i]
        total, result = window_sum, window_sum
        for i in range(m, n):
            # 计算数组总和
            total += cardPoints[i]
            # 入 + 出
            window_sum += cardPoints[i] - cardPoints[i - m]
            # 更新答案
            result = min(result, window_sum)
        return total - result
