# 一、题目描述

## **问题概括**

给定一个整数数组 `nums`，其中每个元素代表一名学生的分数，以及一个正整数 `k`。你需要从数组中选择任意 `k` 个分数（即 `k` 个学生的分数），使得这 `k` 个分数中最高分与最低分之差（极差）尽可能小。返回这个可能的最小差值。

## **示例说明**

- **示例1**：`nums = [90]`, `k = 1`，只有一个分数，极差为 0。
- **示例2**：`nums = [9,4,1,7]`, `k = 2`，选择分数 9 和 7 时极差为 2，是所有组合中的最小值。

## **数据范围**

- `1 <= k <= nums.length <= 1000`
- `0 <= nums[i] <= 10^5`



# 二、解题思路\解题过程

这是今天的“每日一题”，今天这道题目的难度是“简单”，刚开始没有理解好题意，第一次提交报错了，后续仔细分析题意，发现也是一道“滑动窗口”类的题目，不过需要先对题目提供的整个数组进行排序，然后才采用滑动窗口类的经典三步法，当然，具体的代码实现有所不同，可以采用`list`集合，也可以不使用`list`集合，代码如下！

# 三、代码演示

## Java代码

```Java
// 采用list集合
class Solution {
    public int minimumDifference(int[] nums, int k) {
        int result = Integer.MAX_VALUE, n = nums.length;
        List<Integer> list = new ArrayList<>();
        Arrays.sort(nums);
        for (int i = 0; i < n; i++) {
            // 入
            // 右端点进入滑动窗口
            list.add(nums[i]);

            // 更新答案
            int left = i - k + 1;
            if (left < 0) {
                continue;
            }
            result = Math.min(result, Collections.max(list) - Collections.min(list));

            // 出
            list.remove(0);
        }
        return result;
    }
}

// 不采用list集合
class Solution {
    public int minimumDifference(int[] nums, int k) {
        int result = Integer.MAX_VALUE, n = nums.length;
        Arrays.sort(nums);
        // 滑动窗口：[i, i + k - 1]
        for (int i = 0; i + k - 1 < n; i++) {
            int min = nums[i], max = nums[i + k - 1];
            result = Math.min(result, max - min);
        }
        return result;
    }
}
```

```Python
# 采用[]
class Solution:
    def minimumDifference(self, nums: List[int], k: int) -> int:
        result, n, arr = float("inf"), len(nums), []
        nums.sort()
        for i in range(n):
            # 入
            # 右端点进入窗口
            arr.append(nums[i])

            # 更新答案
            left = i - k + 1
            if left < 0:
                continue
            result = min(result, max(arr) - min(arr))

            # 出
            arr.pop(0)
        return result

# 不采用[]
class Solution:
    def minimumDifference(self, nums: List[int], k: int) -> int:
        result, n, i = float("inf"), len(nums), 0
        nums.sort()
        # 滑动窗口：[i, i + k - 1]
        while i + k - 1 < n:
            min_val, max_val = nums[i], nums[i + k - 1]
            result = min(result, max_val - min_val)
            i += 1
        return result
```

## C++代码

```C++
// 采用list集合
class Solution {
public:
    int minimumDifference(vector<int>& nums, int k) {
        int result = INT_MAX, n = nums.size();
        vector<int> arr;
        sort(nums.begin(), nums.end());
        for (int i = 0; i < n; i++) {
            // 入
            // 右端点进入滑动窗口
            arr.push_back(nums[i]);

            // 更新答案
            int left = i - k + 1;
            if (left < 0) {
                continue;
            }
            int maxVal = *max_element(arr.begin(), arr.end());
            int minVal = *min_element(arr.begin(), arr.end());
            result = min(result, maxVal - minVal);

            // 出
            arr.erase(arr.begin());
        }
        return result;
    }
};
```
