# 一、题目描述

## **问题概括**

给定一个二叉树的根节点 `root`。假设根节点位于第 1 层，其子节点位于第 2 层，依此类推。你需要计算每一层节点值之和，并找出总和最大的那一层。如果有多层的总和相同且都是最大，则返回其中最小的层号。

## **示例说明**

- **示例1**：二叉树 `[1,7,0,7,-8,null,null]` 中，第 1 层和为 1，第 2 层和为 7+0=7，第 3 层和为 7+(-8)=-1。最大和为 7 出现在第 2 层，因此返回层号 2。

- **示例2**：更复杂的树中，最大和同样出现在第 2 层。

## **数据范围**

- 树中节点数目在 `[1, 10^4]` 范围内。
- 节点值在 `-10^5` 到 `10^5` 之间。



# 二、解题思路\解题过程

这道题是二叉树遍历的题目，我对二叉树目前只有理论上的了解，对于其代码，我还是没有什么太多的练习，所以，我先去做了[102. 二叉树的层序遍历 - 力扣（LeetCode）](https://leetcode.cn/problems/binary-tree-level-order-traversal/description/)这道题目，用来熟悉熟悉二叉树相关的代码，请看下一题！
这一题的思路如下：

这道题其实是在[102. 二叉树的层序遍历 - 力扣（LeetCode）](https://leetcode.cn/problems/binary-tree-level-order-traversal/description/)这道题上进行扩展的，这102道题需要层序遍历题目给的二叉树，而这第1161道题目则是求层序遍历之后哪一层的总和最大，并最终返回总和最大的那一层的层号（层号从`1`开始）！

所以，只要在第102道题的代码上进行对应的修改，即可得到最终的代码！

代码如下：
```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public int maxLevelSum(TreeNode root) {
        List<TreeNode> qRoot = new ArrayList<>();
        qRoot.add(root);
        int level = 1, maxSums = Integer.MIN_VALUE;
        for(int i = level; !qRoot.isEmpty(); i++){
            List<TreeNode> tmp = qRoot;
            qRoot = new ArrayList<>();
            int sums = 0;
            for(TreeNode node : tmp){
                sums += node.val;
                if(node.left != null){
                    qRoot.add(node.left);
                }
                if(node.right != null){
                    qRoot.add(node.right);
                }
            }
            if(sums > maxSums){
                maxSums = sums;
                level = i;
            }
        }
        return level;
    }
}
```

## Python代码

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxLevelSum(self, root: Optional[TreeNode]) -> int:
        level_sums, level, current = [], 1, [root]
        while current :
            level_sums.append(sum(node.val for node in current))
            nxt = []
            for node in current :
                if node.left : nxt.append(node.left)
                if node.right : nxt.append(node.right)
            current = nxt
        sums_max = max(level_sums)
        for i in range(len(level_sums)) :
            if sums_max == level_sums[i] :
                return i + 1
```

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    int maxLevelSum(TreeNode* root) {
        vector<TreeNode*> qRoot = {root};
        int level = 1, maxSums = INT_MIN;
        for(int i = level; !qRoot.empty(); i++){
            auto tmp = qRoot;
            qRoot.clear();
            int sums = 0;
            for(auto node : tmp){
                sums += node->val;
                if(node->left){
                    qRoot.push_back(node->left);
                }
                if(node->right){
                    qRoot.push_back(node->right);
                }
            }
            if(sums > maxSums){
                maxSums = sums;
                level = i;
            }
        }
        return level;
    }
};
