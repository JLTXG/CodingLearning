# 一、题目描述

## **问题概括**

给定一个仅由字符 '0' 和 '1' 组成的字符串 `s`，长度至少为 2。你需要将 `s` 分割成两个非空的连续子字符串（左子串和右子串）。分割的得分定义为：左子串中字符 '0' 的个数加上右子串中字符 '1' 的个数。请计算所有可能分割方式中能获得的最大得分，并返回该值。

## **示例说明**

- **示例1**：`s = "011101"`，最大得分为 5，对应的分割是左子串为 "0"，右子串为 "11101"（得分为 1+4=5）。
- **示例2**：`s = "00111"`，最大得分为 5，对应的分割是左子串为 "00"，右子串为 "111"（得分为 2+3=5）。
- **示例3**：`s = "1111"`，最大得分为 3，对应的分割是左子串为 "111"，右子串为 "1"（得分为 0+3=3）。

## **数据范围**

- 字符串长度在 2 到 500 之间。
- 字符串仅包含 '0' 和 '1'。



# 二、解题思路\解题过程

这又是一道简单的编程题目，可以采用两种思路和解法来进行求解：
## 第一种：暴力求解

最朴素的想法：
- 枚举每一个分割点 `i`（从 0 到 n-2）
- 对每个 `i`，遍历左边统计 `'0'`，遍历右边统计 `'1'`
- 计算得分，更新最大值

代码如下：

```java
class Solution {
    public int maxScore(String s) {
        int result = 0, n = s.length();
        for(int i = 0; i < n - 1; i++){
            int flag = 0;
            for(int a = 0; a <= i; a++){
                if(s.charAt(a) == '0'){
                    flag++;
                }
            }
            for(int b = i + 1; b < n; b++){
                if(s.charAt(b) == '1'){
                    flag++;
                }
            }
            result = result > flag ? result : flag;
        }
        return result;
    }
}
```

## 第二种：贪心 + 前缀思想

我们换一个角度思考：

> **初始状态**：假设分割点在最左边（即左子串为空，右子串为整个字符串）  
> 此时得分 = 0（左无0） + `totalOnes`（右全1） = `totalOnes`

然后，我们**从左到右移动分割点**，每次将一个字符从“右边”移到“左边”，并**动态更新得分**：

- 如果移过去的字符是 `'0'`：
    - 左边 `'0'` 数量 +1 → **总得分 +1**
- 如果移过去的字符是 `'1'`：
    - 右边 `'1'` 数量 -1 → **总得分 -1**

代码如下：
```cpp
class Solution {
public:
    int maxScore(string s) {
        int n = s.length(), result = 0, flag = 0;
        for(int i = 0; i < n; i++){
            if(s[i] == '1') flag++;
        }
        for(int i = 0; i < n - 1; i++){
            if(s[i] == '1') {
                flag--;
            } else {
                flag++;
            }
            result = result > flag ? result : flag;
        }
        return result;
    }
};
```

    def maxScore(self, s: str) -> int:
        result, n = 0, len(s)
        for i in range(n - 1):
            flag = 0
            for a in range(i + 1):
                if s[a] == '0' :
                    flag += 1
            for b in range(i + 1, n):
                if s[b] == '1' :
                    flag += 1

        return result
