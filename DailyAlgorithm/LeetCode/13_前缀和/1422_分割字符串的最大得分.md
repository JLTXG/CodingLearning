>标签：字符串  |  前缀和

# 一、题目描述

给你一个由若干 0 和 1 组成的字符串 `s` ，请你计算并返回将该字符串分割成两个 **非空** 子字符串（即 **左** 子字符串和 **右** 子字符串）所能获得的最大得分。

「分割字符串的得分」为 **左** 子字符串中 **0** 的数量加上 **右** 子字符串中 **1** 的数量。

 

**示例 1：**

```
输入：s = "011101"
输出：5 
解释：
将字符串 s 划分为两个非空子字符串的可行方案有：
左子字符串 = "0" 且 右子字符串 = "11101"，得分 = 1 + 4 = 5 
左子字符串 = "01" 且 右子字符串 = "1101"，得分 = 1 + 3 = 4 
左子字符串 = "011" 且 右子字符串 = "101"，得分 = 1 + 2 = 3 
左子字符串 = "0111" 且 右子字符串 = "01"，得分 = 1 + 1 = 2 
左子字符串 = "01110" 且 右子字符串 = "1"，得分 = 2 + 1 = 3
```

**示例 2：**

```
输入：s = "00111"
输出：5
解释：当 左子字符串 = "00" 且 右子字符串 = "111" 时，我们得到最大得分 = 2 + 3 = 5
```

**示例 3：**

```
输入：s = "1111"
输出：3
```

 

**提示：**

- `2 <= s.length <= 500`
- 字符串 `s` 仅由字符 `'0'` 和 `'1'` 组成。



# 二、解题思路\解题过程

这又是一道简单的编程题目，可以采用两种思路和解法来进行求解：
## 第一种：暴力求解

最朴素的想法：
- 枚举每一个分割点 `i`（从 0 到 n-2）
- 对每个 `i`，遍历左边统计 `'0'`，遍历右边统计 `'1'`
- 计算得分，更新最大值
代码如下：
```java
class Solution {
    public int maxScore(String s) {
        int result = 0, n = s.length();
        for(int i = 0; i < n - 1; i++){
            int flag = 0;
            for(int a = 0; a <= i; a++){
                if(s.charAt(a) == '0'){
                    flag++;
                }
            }
            for(int b = i + 1; b < n; b++){
                if(s.charAt(b) == '1'){
                    flag++;
                }
            }
            result = result > flag ? result : flag;
        }
        return result;
    }
}
```

## 第二种：贪心 + 前缀思想

我们换一个角度思考：

> **初始状态**：假设分割点在最左边（即左子串为空，右子串为整个字符串）  
> 此时得分 = 0（左无0） + `totalOnes`（右全1） = `totalOnes`

然后，我们**从左到右移动分割点**，每次将一个字符从“右边”移到“左边”，并**动态更新得分**：

- 如果移过去的字符是 `'0'`：
    - 左边 `'0'` 数量 +1 → **总得分 +1**
- 如果移过去的字符是 `'1'`：
    - 右边 `'1'` 数量 -1 → **总得分 -1**

代码如下：
```cpp
class Solution {
public:
    int maxScore(string s) {
        int n = s.length(), result = 0, flag = 0;
        for(int i = 0; i < n; i++){
            if(s[i] == '1') flag++;
        }
        for(int i = 0; i < n - 1; i++){
            if(s[i] == '1') {
                flag--;
            } else {
                flag++;
            }
            result = result > flag ? result : flag;
        }
        return result;
    }
};
```

        result, n = 0, len(s)

            flag = 0
            for a in range(i + 1):
                if s[a] == '0' :
                    flag += 1

        return result
