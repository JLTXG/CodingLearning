# 一、题目描述

给你两个整数 `num1` 和 `num2`，返回这两个整数的和。

 

**示例 1：**

```
输入：num1 = 12, num2 = 5
输出：17
解释：num1 是 12，num2 是 5 ，它们的和是 12 + 5 = 17 ，因此返回 17 。
```

**示例 2：**

```
输入：num1 = -10, num2 = 4
输出：-6
解释：num1 + num2 = -6 ，因此返回 -6 。
```

 

**提示：**

- `-100 <= num1, num2 <= 100`



# 二、解题思路\解题过程

这是一道非常简单的题目，就是将题目给的两个整数相加，然后在返回即可，而且，也不用考虑==溢出==的问题，因为题目给出的整数的范围为`-100 <= num1, num2 <= 100`，代码如下：

```java
class Solution {
    public int sum(int num1, int num2) {
        return num1 + num2;
    }
}
```
