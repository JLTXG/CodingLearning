# 一、题目描述

## **问题概括**

给定一个正整数 `n`，计算并返回 `n` 的十进制表示中各位数字的乘积与各位数字之和的差值。

## **示例说明**

- **示例1**：`n = 234`，各位数字之积 = 2×3×4=24，各位数字之和 = 2+3+4=9，差值 = 24-9=15。
- **示例2**：`n = 4421`，各位数字之积 = 4×4×2×1=32，各位数字之和 = 4+4+2+1=11，差值 = 32-11=21。

## **数据范围**

- `1 <= n <= 10^5`



# 二、解题思路\解题过程

这也是一道简单的数学题目，只需要按照题目的意思模拟就好了，代码如下：

```cpp
class Solution {
public:
    int subtractProductAndSum(int n) {
        int resultSum = 0, resultPro = 1;
        while(n != 0){
            resultSum += n % 10;
            resultPro *= n % 10;
            n /= 10;
        }
        return resultPro - resultSum;
    }
};
```
