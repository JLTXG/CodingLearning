# 一、题目描述

给你一个整数数组 `nums` 。

如果一组数字 `(i,j)` 满足 `nums[i]` == `nums[j]` 且 `i` < `j` ，就可以认为这是一组 **好数对** 。

返回好数对的数目。

 

**示例 1：**

```
输入：nums = [1,2,3,1,1,3]
输出：4
解释：有 4 组好数对，分别是 (0,3), (0,4), (3,4), (2,5) ，下标从 0 开始
```

**示例 2：**

```
输入：nums = [1,1,1,1]
输出：6
解释：数组中的每组数字都是好数对
```

**示例 3：**

```
输入：nums = [1,2,3]
输出：0
```

 

**提示：**

- `1 <= nums.length <= 100`
- `1 <= nums[i] <= 100`



# 二、解题思路\解题过程

这道题目也是一道简单的题目，可以有两种解法，这道题按照正常的思路写就OK了，没有什么套路在里面，如下：

- 第一种：采用“桶排序”的思想来求解，代码如下：
```java
class Solution {
    public int numIdenticalPairs(int[] nums) {
        int result = 0, n = nums.length;
        int[] arr = new int[101];
        for (int i = 0; i < n; i++) {
            arr[nums[i]]++;
        }
        for (int i = 1; i < 101; i++){
            if(arr[i] > 1){
                for(int j = 1; j < arr[i]; j++){
                    result += j;
                }
            }
        }
        return result;
    }
}
```

# 桶排序思想
class Solution:
    def numIdenticalPairs(self, nums: List[int]) -> int:
        result, n = 0, len(nums)
        arr = [0] * 101
        for i in range(n):
            arr[nums[i]] += 1
        for i in range(1, 101):
            if arr[i] > 1 :
                for j in range(1, arr[i]):
                    result += j
        return result
