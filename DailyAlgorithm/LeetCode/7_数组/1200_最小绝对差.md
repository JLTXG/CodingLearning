# 一、题目描述

## **问题概括**

给定一个元素互不相同的整数数组 `arr`，请找出所有满足以下条件的元素对 `(a, b)`：

- `a` 和 `b` 均为数组中的元素，且 `a < b`；
- `b` 与 `a` 的差值等于数组中任意两个不同元素的最小绝对差。

将这些元素对按照升序顺序返回（即先按 `a` 升序，若 `a` 相同则按 `b` 升序）。

## **示例说明**

- **示例1**：`arr = [4,2,1,3]`，最小差值为 1，结果包含所有相邻差为 1 的元素对：`[[1,2],[2,3],[3,4]]`。
- **示例2**：`arr = [1,3,6,10,15]`，最小差值为 2，只有 `[1,3]` 满足，返回 `[[1,3]]`。
- **示例3**：`arr = [3,8,-10,23,19,-4,-14,27]`，最小差值为 4，返回 `[[-14,-10],[19,23],[23,27]]`。

## **数据范围**

- `2 <= arr.length <= 10^5`
- `-10^6 <= arr[i] <= 10^6`



# 二、解题思路\解题过程

这是今天的“每日一题”，挺简单的，就纯模拟，先将题目提供的数组进行排序（升序），然后循环一遍记录元素对之间最小的绝对差，然后在创建对应的变量值——`List<List<Integer>> result = new ArrayList<>();`，创建完之后，再次循环升序后的数组，记录哪两个元素之间的绝对差等于最小绝对差，如果等于，就将这两个元素组成元素对——只有两个元素的一维数组——然后在存入`result`中，再次遍历完整个数组之后，就可以返回了，代码如下！

# 三、代码演示

## Java代码

```Java
class Solution {
    public List<List<Integer>> minimumAbsDifference(int[] arr) {
        Arrays.sort(arr);
        int minVal = Integer.MAX_VALUE, n = arr.length;
        List<List<Integer>> result = new ArrayList<>();
        for (int i = 0; i < n - 1; i++) {
            int val = arr[i + 1] - arr[i];
            if (val < minVal) {
                minVal = val;
                result.clear();
                result.add(Arrays.asList(arr[i], arr[i + 1]));
            } else if (val == minVal) {
                result.add(Arrays.asList(arr[i], arr[i + 1]));
            }
        }
        return result;
    }
}
```

```Python
# 简洁版
class Solution:
    def minimumAbsDifference(self, arr: List[int]) -> List[List[int]]:
        arr.sort()
        min_val = min(arr[i + 1] - arr[i] for i in range(len(arr) - 1))
        return [
            [arr[i], arr[i + 1]]
            for i in range(len(arr) - 1)
            if arr[i + 1] - arr[i] == min_val
        ]
        
```

```C++
                result.push_back({arr[i], arr[i + 1]});
```
