# 一、题目描述

## **问题概括**

给定一个长度为 `2 * n` 的整数数组 `nums`。数组中的元素按特定顺序分为两段：前 `n` 个元素（记为 `x1, x2, ..., xn`）和后 `n` 个元素（记为 `y1, y2, ..., yn`）。

你需要按照 `[x1, y1, x2, y2, ..., xn, yn]` 的顺序，将这些元素重新排列并返回新的数组。

## **示例说明**

- **示例1**：输入 `nums = [2,5,1,3,4,7]`, `n = 3`。重排后结果为 `[2,3,5,4,1,7]`。
- **示例2**：输入 `nums = [1,2,3,4,4,3,2,1]`, `n = 4`。重排后结果为 `[1,4,2,3,3,2,4,1]`。
- **示例3**：输入 `nums = [1,1,2,2]`, `n = 2`。重排后结果为 `[1,2,1,2]`。

## **数据范围**

- `1 <= n <= 500`
- `nums.length == 2 * n`
- `1 <= nums[i] <= 10^3`



# 二、解题思路\解题过程

这道题也是一道简单的编程题目，直接采用模拟思路来写就好了，没有什么技术含量，只需要关注数组的索引是否超出范围就好了，一步步找好规律就可以求解出来，代码如下：
```cpp
class Solution {
public:
    vector<int> shuffle(vector<int>& nums, int n) {
        int flag = 0;
        vector<int> arr(2 * n);
        for (int i = 0, j = n; i < n && j < 2 * n; i++, j++) {
            arr[flag] = nums[i];
            arr[flag + 1] = nums[j];
            flag += 2;
        }
        return arr;
    }
};
```

```Python
class Solution:
    def shuffle(self, nums: List[int], n: int) -> List[int]:
        flag, arr = 0, [0] * 2 * n
        for i in range(n):
            arr[flag] = nums[i]
            arr[flag + 1] = nums[i + n]
            flag += 2
        return arr
```
