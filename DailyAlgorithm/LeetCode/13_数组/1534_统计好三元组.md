# 一、题目描述

## **问题概括**

给定一个整数数组 `arr` 和三个非负整数阈值 `a`、`b`、`c`。一个“好三元组”由三个下标 `(i, j, k)` 组成，满足：

1. **下标顺序**：`0 <= i < j < k < arr.length`（即三个下标严格递增）。
2. **差值约束**：
    - `arr[i]` 与 `arr[j]` 之差的绝对值不超过 `a`。
    - `arr[j]` 与 `arr[k]` 之差的绝对值不超过 `b`。
    - `arr[i]` 与 `arr[k]` 之差的绝对值不超过 `c`。

你需要统计数组中所有满足上述条件的三元组数量。

## **示例说明**

- **示例1**：`arr = [3,0,1,1,9,7]`, `a = 7`, `b = 2`, `c = 3`。满足条件的三元组有 4 个，具体为 `(3,0,1)`（对应下标 `(0,1,2)`）、`(3,0,1)`（对应下标 `(0,1,3)`）、`(3,1,1)`（对应下标 `(0,2,3)`）、`(0,1,1)`（对应下标 `(1,2,3)`），因此输出 `4`。
- **示例2**：`arr = [1,1,2,2,3]`, `a = 0`, `b = 0`, `c = 1`。没有三元组能同时满足三个差值约束，因此输出 `0`。

## **数据范围**

- 数组长度在 `3` 到 `100` 之间。
- 数组元素值在 `0` 到 `1000` 之间。
- `a`、`b`、`c` 的值在 `0` 到 `1000` 之间。



# 二、解题思路\解题过程

这道题有多种解法，今天由于时间原因，只能先用最简单的暴力求解的方法，后续有时间，或者通过其他题目来熟悉其他解法——比如灵神题解中的`前缀和`、`排序+枚举中间+三指针`等等方法，暴力求解的代码如下：

```java
class Solution {
    public int countGoodTriplets(int[] arr, int a, int b, int c) {
        int result = 0, n = arr.length;
        for(int i = 0; i < n; i++){
            for(int j = i + 1; j < n; j++){
                for(int k = j + 1; k < n; k++){
                    if(Math.abs(arr[i] - arr[j]) <= a && Math.abs(arr[j] - arr[k]) <= b && Math.abs(arr[i] - arr[k]) <= c){
                        result++;
                    }
                }
            }
        }
        return result;
    }
}
```

class Solution:
    def countGoodTriplets(self, arr: List[int], a: int, b: int, c: int) -> int:
        result, n = 0, len(arr)
        for i in range(n):
            for j in range(i + 1, n):
                for k in range(j + 1, n):
                    if abs(arr[i] - arr[j]) <= a and abs(arr[j] - arr[k]) <= b and abs(arr[i] - arr[k]) <= c :
                        result += 1
        return result
