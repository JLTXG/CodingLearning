# 一、题目描述

## **问题概括**

给定一个字符串数组 `words`，以及一个下标闭区间 `[left, right]`。如果一个非空字符串**同时满足**以下两个条件，则称其为一个 **元音字符串**：

1. 以元音字母开头。
2. 以元音字母结尾。

其中，元音字母仅指 `'a'`, `'e'`, `'i'`, `'o'`, `'u'` 这五个小写字母。

请你统计并返回数组 `words` 中，所有下标 `i` 在给定区间 `[left, right]` 内（包含两端）的字符串里，属于元音字符串的个数。

## **示例说明**

- **示例1**：`words = ["are","amy","u"]`, `left = 0`, `right = 2`。
    - `"are"` 以 `'a'` 开头、以 `'e'` 结尾，是元音字符串。
    - `"amy"` 以 `'a'` 开头但以 `'y'` 结尾，不是元音字符串。
    - `"u"` 以 `'u'` 开头并以 `'u'` 结尾，是元音字符串。
    - 因此，在区间 `[0, 2]` 内，共有 `2` 个元音字符串。
- **示例2**：`words = ["hey","aeo","mu","ooo","artro"]`, `left = 1`, `right = 4`。
    - 区间内字符串 `"aeo"`, `"mu"`, `"ooo"`, `"artro"` 中，`"aeo"`, `"ooo"`, `"artro"` 符合条件，共有 `3` 个元音字符串。

## **数据范围**

- `1 <= words.length <= 1000`
- `1 <= words[i].length <= 10`
- `words[i]` 仅由小写英文字母组成。
- `0 <= left <= right < words.length`



# 二、解题思路\解题过程

同样的，这也是一道简单的编程题目，同样“简单题目，简单做”，直接用最朴素的方法来姐这道题目，代码如下：
```cpp
class Solution {
public:
    int vowelStrings(vector<string>& words, int left, int right) {
        int result = 0;
        for(int i = left; i <= right; i++){
            int n = words[i].length();
            if((words[i][0] == 'a' || words[i][0] == 'e' || words[i][0] == 'i' || words[i][0] == 'o' || words[i][0] == 'u') && (words[i][n - 1] == 'a' || words[i][n - 1] == 'e' || words[i][n - 1] == 'i' || words[i][n - 1] == 'o' || words[i][n - 1] == 'u')) {
                result++;
            }
        }
        return result;
    }
};
```
