# 一、题目描述

## **问题概括**

给定一个整数数组 `nums`。对于数组中的两个下标 `i` 和 `j`，如果满足 `i < j` 且 `nums[i]` 与 `nums[j]` 相等，则称 `(i, j)` 为一个“好数对”。你的任务是统计数组中所有这样的好数对的总数量。

## **示例说明**

- **示例1**：`nums = [1,2,3,1,1,3]`。好数对有 `(0,3)`、`(0,4)`、`(3,4)`、`(2,5)`，共 4 对。
- **示例2**：`nums = [1,1,1,1]`。所有可能的数对（共6对）都是好数对，因此输出 6。
- **示例3**：`nums = [1,2,3]`。没有相等的元素对，因此输出 0。

## **数据范围**

- 数组长度在 `1` 到 `100` 之间。
- 数组中的每个元素值在 `1` 到 `100` 之间。



# 二、解题思路\解题过程

这道题目也是一道简单的题目，可以有两种解法，这道题按照正常的思路写就OK了，没有什么套路在里面，如下：

- 第一种：采用“桶排序”的思想来求解，代码如下：
```java
class Solution {
    public int numIdenticalPairs(int[] nums) {
        int result = 0, n = nums.length;
        int[] arr = new int[101];
        for (int i = 0; i < n; i++) {
            arr[nums[i]]++;
        }
        for (int i = 1; i < 101; i++){
            if(arr[i] > 1){
                for(int j = 1; j < arr[i]; j++){
                    result += j;
                }
            }
        }
        return result;
    }
}
```

# 桶排序思想
class Solution:
    def numIdenticalPairs(self, nums: List[int]) -> int:
        result, n = 0, len(nums)
        arr = [0] * 101
        for i in range(n):
            arr[nums[i]] += 1
        for i in range(1, 101):
            if arr[i] > 1 :
                for j in range(1, arr[i]):
                    result += j
        return result
