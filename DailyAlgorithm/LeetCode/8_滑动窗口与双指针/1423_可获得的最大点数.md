[1423. 可获得的最大点数 - 力扣（LeetCode）](https://leetcode.cn/problems/maximum-points-you-can-obtain-from-cards/description/)

>标签：数组  |  前缀和  |  滑动窗口

# 一、题目描述

几张卡牌 **排成一行**，每张卡牌都有一个对应的点数。点数由整数数组 `cardPoints` 给出。

每次行动，你可以从行的开头或者末尾拿一张卡牌，最终你必须正好拿 `k` 张卡牌。

你的点数就是你拿到手中的所有卡牌的点数之和。

给你一个整数数组 `cardPoints` 和整数 `k`，请你返回可以获得的最大点数。

 

**示例 1：**

```
输入：cardPoints = [1,2,3,4,5,6,1], k = 3
输出：12
解释：第一次行动，不管拿哪张牌，你的点数总是 1 。但是，先拿最右边的卡牌将会最大化你的可获得点数。最优策略是拿右边的三张牌，最终点数为 1 + 6 + 5 = 12 。
```

**示例 2：**

```
输入：cardPoints = [2,2,2], k = 2
输出：4
解释：无论你拿起哪两张卡牌，可获得的点数总是 4 。
```

**示例 3：**

```
输入：cardPoints = [9,7,7,9,7,7,9], k = 7
输出：55
解释：你必须拿起所有卡牌，可以获得的点数为所有卡牌的点数之和。
```

**示例 4：**

```
输入：cardPoints = [1,1000,1], k = 1
输出：1
解释：你无法拿到中间那张卡牌，所以可以获得的最大点数为 1 。 
```

**示例 5：**

```
输入：cardPoints = [1,79,80,1,1,1,200,1], k = 3
输出：202
```

 

**提示：**

- `1 <= cardPoints.length <= 10^5`
- `1 <= cardPoints[i] <= 10^4`
- `1 <= k <= cardPoints.length`



# 二、解题思路\解题过程

这也是一道“滑动窗口”类的题目，这道题目可以用两种不同的思维方式来求解——逆向思维+正向思维，具体内容如下所示：
## 逆向思维

题目要从拿走`k`张卡牌，总共有`n`张卡牌，那么就只剩下`n - k`张卡牌了，另外，这个`k`张卡牌只能从首或者从尾开始拿，在拿够`k`张卡牌之后，剩下的`n - k`也必然是连续的卡牌，那么接下来，整个题目就转换成了`求数组中连续 n - k 个连续子数组和的最小值`。

在转换题目意思后，就可以采用灵神总结的“滑动窗口”类的经典解题方法——“**入——更新——出**”——就可以求出最终的答案了，代码如下：

```cpp
class Solution {
public:
    int maxScore(vector<int>& cardPoints, int k) {
        int windowSum = 0, n = cardPoints.size();
        int m = n - k;
        // 初始化第一个窗口
        for (int i = 0; i < m; i++) {
            windowSum += cardPoints[i];
        }

        int sum = windowSum, result = windowSum;
        for (int i = m; i < n; i++) {
            // 计算数组的总和
            sum += cardPoints[i];
            // 入 + 出
            windowSum += cardPoints[i] - cardPoints[i - m];
            // 更新答案
            result = min(result, windowSum);
        }
        
        return sum - result;
    }
};
```

## 正向思维

我们也可以采用正向思维来进行求解，正向思维也很简单，题目要求的卡牌只能从“首”或者“尾”进行拿取：
- 可以只拿首部的`k`张
- 可以只拿尾部的`k`张
- 可以先从首部开始拿，然后再从尾部拿
- 可以先从尾部开始拿，然后再从首部拿

所以，我们简单点，先从首部开始，求出第一个窗口内的总和，然后减去首部窗口内的对应数组值，然后在加上尾部的值，逐步更新，求出题目要求的`k`张卡牌值的最大总和！

代码如下：

```java
class Solution {
    public int maxScore(int[] cardPoints, int k) {
        int windowSum = 0, n = cardPoints.length;
        // 初始化第一个窗口
        for (int i = 0; i < k; i++) {
            windowSum += cardPoints[i];
        }

        int result = windowSum;
        for (int i = 1; i <= k; i++) {
            // 入 + 出
            windowSum += cardPoints[n - i] - cardPoints[k - i];
            // 更新答案
            result = Math.max(result, windowSum);
        }
        
        return result;
    }
}
```

## Python代码

```Python
# 逆向思维
class Solution:
    def maxScore(self, cardPoints: List[int], k: int) -> int:
        window_sum, n = 0, len(cardPoints)
        m = n - k
        # 初始化第一个窗口
        for i in range(m):
            window_sum += cardPoints[i]
        total, result = window_sum, window_sum
        for i in range(m, n):
            # 计算数组总和
            total += cardPoints[i]
            # 入 + 出
            window_sum += cardPoints[i] - cardPoints[i - m]
            # 更新答案
            result = min(result, window_sum)
        return total - result
```
