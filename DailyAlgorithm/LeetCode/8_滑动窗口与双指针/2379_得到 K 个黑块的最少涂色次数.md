[2379. 得到 K 个黑块的最少涂色次数 - 力扣（LeetCode）](https://leetcode.cn/problems/minimum-recolors-to-get-k-consecutive-black-blocks/description/)

>标签：字符串  |  滑动窗口

# 一、题目描述

给你一个长度为 `n` 下标从 **0** 开始的字符串 `blocks` ，`blocks[i]` 要么是 `'W'` 要么是 `'B'` ，表示第 `i` 块的颜色。字符 `'W'` 和 `'B'` 分别表示白色和黑色。

给你一个整数 `k` ，表示想要 **连续** 黑色块的数目。

每一次操作中，你可以选择一个白色块将它 **涂成** 黑色块。

请你返回至少出现 **一次** 连续 `k` 个黑色块的 **最少** 操作次数。

 

**示例 1：**

```
输入：blocks = "WBBWWBBWBW", k = 7
输出：3
解释：
一种得到 7 个连续黑色块的方法是把第 0 ，3 和 4 个块涂成黑色。
得到 blocks = "BBBBBBBWBW" 。
可以证明无法用少于 3 次操作得到 7 个连续的黑块。
所以我们返回 3 。
```

**示例 2：**

```
输入：blocks = "WBWBBBW", k = 2
输出：0
解释：
不需要任何操作，因为已经有 2 个连续的黑块。
所以我们返回 0 。
```

 

**提示：**

- `n == blocks.length`
- `1 <= n <= 100`
- `blocks[i]` 要么是 `'W'` ，要么是 `'B'` 。
- `1 <= k <= n`



# 二、解题思路\解题过程

这是今天做的“滑动窗口”类的题目，同样采用的是灵神总结的“滑动窗口”类题目的经典总结方法，再次回顾一下：

窗口右端点在`i`时，由于窗口长度为`k`，所以窗口在左端点为`i - k + 1`。

一共有三步：`入——更新——出`。

1. **入**：下标为`i`的元素进入窗口，更新相关统计量。如果窗口左端点`i - k + 1 < 0`，则尚未形成第一个窗口，重复第一步。
2. **更新**：更新答案。一般是更新最大值/最小值。
3. **出**：下标为`i - k + 1`的元素离开窗口，更新相关统计量，为下一个循环做准备。

以上三步适用于所有定长滑窗题目。

但是，这道题需要稍微改一下，我们需要先初始化完成第一个窗口，得到第一个窗口中的数值，后续的更新值也是从上一个值来进行更改的，所以，我们需要初始化完成第一个窗口，然后，接下来的滑动窗口中的顺序为“入——出——更新”，这里的顺序与上述提到的顺序不一样，所以，这个部分我们得根据题目的要求来进行调整，才能写出正确的代码！



# 三、代码演示

## Java代码

```Java
class Solution {
    public int minimumRecolors(String blocks, int k) {
        char[] arr = blocks.toCharArray();
        int flag = 0, n = arr.length;

        // 初始化第一个窗口
        for (int i = 0; i < k; i++) {
            if (arr[i] == 'W') {
                flag++;
            }
        }

        // 滑动窗口：从第 k 个字符开始向右滑动
        int result = flag;
        for (int i = k; i < n; i++) {
            // 加入新字符 arr[i]
            if (arr[i] == 'W') {
                flag++;
            }

            // 出
            if (arr[i - k] == 'W') {
                flag--;
            }

            // 更新答案
            result = Math.min(result, flag);
        }
        return result;
    }
}
```



## Python代码

```Python
class Solution:
    def minimumRecolors(self, blocks: str, k: int) -> int:
        flag, result, n = 0, 0, len(blocks)
        # 初始化第一个窗口
        for i in range(k):
            if blocks[i] == 'W' :
                flag += 1
        
        # 滑动窗口
        result = flag
        for i in range(k, n):
            # 入
            if blocks[i] == 'W' :
                flag += 1
            
            # 出
            if blocks[i - k] == 'W' :
                flag -= 1
            
            # 更新答案
            result = flag if result > flag else result

        return result
```
