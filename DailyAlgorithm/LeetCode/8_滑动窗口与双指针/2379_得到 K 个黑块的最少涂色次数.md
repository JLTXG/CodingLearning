# 一、题目描述

## **问题概括**

给定一个长度为 `n` 的字符串 `blocks`，其中每个字符要么是 `'W'`（白色），要么是 `'B'`（黑色）。你希望在该字符串中得到**至少一段**长度为 `k` 的连续黑色块。每次操作允许你将任意一个白色块涂成黑色。

你需要计算并返回达成目标所需的**最少操作次数**。

## **示例说明**

- **示例1**：`blocks = "WBBWWBBWBW"`, `k = 7`。一种最优方案是将第 0、3、4 个块涂黑，得到 `"BBBBBBBWBW"`，包含连续 7 个黑块，操作次数为 3。
- **示例2**：`blocks = "WBWBBBW"`, `k = 2`。字符串中已经存在连续的黑块，无需操作，结果为 0。

## **数据范围**

- `n == blocks.length`
- `1 <= n <= 100`
- `blocks[i]` 为 `'W'` 或 `'B'`
- `1 <= k <= n`



# 二、解题思路\解题过程

这是今天做的“滑动窗口”类的题目，同样采用的是灵神总结的“滑动窗口”类题目的经典总结方法，再次回顾一下：

窗口右端点在`i`时，由于窗口长度为`k`，所以窗口在左端点为`i - k + 1`。

一共有三步：`入——更新——出`。

1. **入**：下标为`i`的元素进入窗口，更新相关统计量。如果窗口左端点`i - k + 1 < 0`，则尚未形成第一个窗口，重复第一步。
2. **更新**：更新答案。一般是更新最大值/最小值。
3. **出**：下标为`i - k + 1`的元素离开窗口，更新相关统计量，为下一个循环做准备。

以上三步适用于所有定长滑窗题目。

但是，这道题需要稍微改一下，我们需要先初始化完成第一个窗口，得到第一个窗口中的数值，后续的更新值也是从上一个值来进行更改的，所以，我们需要初始化完成第一个窗口，然后，接下来的滑动窗口中的顺序为“入——出——更新”，这里的顺序与上述提到的顺序不一样，所以，这个部分我们得根据题目的要求来进行调整，才能写出正确的代码！



# 三、代码演示

## Java代码

```Java
class Solution {
    public int minimumRecolors(String blocks, int k) {
        char[] arr = blocks.toCharArray();
        int flag = 0, n = arr.length;

        // 初始化第一个窗口
        for (int i = 0; i < k; i++) {
            if (arr[i] == 'W') {
                flag++;
            }
        }

        // 滑动窗口：从第 k 个字符开始向右滑动
        int result = flag;
        for (int i = k; i < n; i++) {
            // 加入新字符 arr[i]
            if (arr[i] == 'W') {
                flag++;
            }

            // 出
            if (arr[i - k] == 'W') {
                flag--;
            }

            // 更新答案
            result = Math.min(result, flag);
        }
        return result;
    }
}
```



## Python代码

```Python
class Solution:
    def minimumRecolors(self, blocks: str, k: int) -> int:
        flag, result, n = 0, 0, len(blocks)
        # 初始化第一个窗口
        for i in range(k):
            if blocks[i] == 'W' :
                flag += 1
        
        # 滑动窗口
        result = flag
        for i in range(k, n):
            # 入
            if blocks[i] == 'W' :
                flag += 1
            
            # 出
            if blocks[i - k] == 'W' :
                flag -= 1
            
            # 更新答案
            result = flag if result > flag else result

        return result
```
