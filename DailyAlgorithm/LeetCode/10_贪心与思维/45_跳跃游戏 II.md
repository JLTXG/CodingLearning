# 一、题目描述

## **问题概括**

你被给予一个整数数组 `nums`，数组中的每个位置 `i` 都有一个数字 `nums[i]`，它代表从该位置能够**向前跳跃的最大距离**。具体来说，当你处在下标 `i` 时，你可以选择跳转到任意一个下标 `i + j`，其中 `j` 的取值范围是 `0` 到 `nums[i]`（包含两端），并且跳转后的位置不能超出数组的边界。

你的起点是数组的第一个位置（下标 `0`），目标是**以最少的跳跃次数**到达数组的最后一个位置（下标 `n-1`）。题目数据保证一定存在一条从起点到终点的可行路径。

## **示例说明**

- **示例1**：数组为 `[2,3,1,1,4]`。一种最优的跳跃路径是：从下标 `0` 跳 `1` 步到下标 `1`（此时跳跃次数为1），然后从下标 `1` 跳 `3` 步直接到达下标 `4`（跳跃次数增加1），总共只需要 `2` 次跳跃。
- **示例2**：数组为 `[2,3,0,1,4]`。同样可以通过 `2` 次跳跃到达终点：先从下标 `0` 跳 `1` 步到下标 `1`，再从下标 `1` 跳 `3` 步到下标 `4`。

## **数据范围**

- 数组的长度 `n` 在 `1` 到 `10^4` 之间。
- 数组中的每个元素值在 `0` 到 `1000` 之间。



# 二、解题思路\解题过程

这道题采用贪心的思想，先定义三个值，`current`表示当前索引到的、可以跳转到的最大位置，`next`表示下一个能跳转到的最大位置，`result`表示最小跳跃次数，上述三个定义的值都为0，然后接下来我们需要去遍历题目给的数组，`next`的值需要取当前索引范围里的最大值，当遍历到的位置与`current`相等时（并且当前的遍历位置不是数组的最后一个位置），说明当前索引范围已经到头了，接下来就需要去“跳转”了，这个时候`result`就需要加`1`了，然后在更新`current`——将上一个索引范围内记录的`next`值赋予`current`。然后一直遍历下去，直到`current`的值`大于等于`数组的最后一个下标值即可跳出循环（当然也可以不加这一段逻辑，直到遍历完整个数组为止！）
代码如下：

```java
class Solution {
    public int jump(int[] nums) {
        //  current表示当前索引到的、可以跳转到的最大位置，next表示下一个能跳转到的最大位置，result表示跳了几次
        int current = 0, next = 0, result = 0, n = nums.length;
        for(int i = 0; i < n; i++){
            next = (i + nums[i]) > next ? (i + nums[i]) : next;
            if(i == current){
                if(i != n - 1){
                    result++;
                    current = next;
                }
                if(current >= n - 1){
                    break;
                }
            }
        }
        return result;
    }
}
```

class Solution:
    def jump(self, nums: List[int]) -> int:
        current, num_next, result, n =0, 0, 0, len(nums)
        for i in range(n):
            num_next = num_next if num_next > i + nums[i] else i + nums[i]
            if i == current :
                if i != n - 1 :
                    result += 1
                    current = num_next
                if current >= n - 1 :
                    break
        return result
