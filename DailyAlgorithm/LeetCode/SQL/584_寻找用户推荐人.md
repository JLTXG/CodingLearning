# 一、题目描述

## **问题概括**

给定一个客户表 `Customer`，包含字段 `id`（客户标识）、`name`（客户姓名）和 `referee_id`（推荐该客户的客户标识，若为空则表示无推荐人）。需要查询出所有满足以下条件之一的客户姓名：

1. 被任意一个 `id` 不等于 2 的客户所推荐（即 `referee_id` 不为空且 `referee_id != 2`）。
2. 没有被任何客户推荐（即 `referee_id` 为空）。

返回结果可以按任意顺序排列。

## 原题详细示例

表: `Customer`

```
+-------------+---------+
| Column Name | Type    |
+-------------+---------+
| id          | int     |
| name        | varchar |
| referee_id  | int     |
+-------------+---------+
在 SQL 中，id 是该表的主键列。
该表的每一行表示一个客户的 id、姓名以及推荐他们的客户的 id。
```

找出以下客户的姓名：

1. **被任何** `id != 2` 的用户推荐。
2. **没有被** 任何用户推荐。

以 **任意顺序** 返回结果表。

结果格式如下所示。

 

**示例 1：**

```
输入： 
Customer 表:
+----+------+------------+
| id | name | referee_id |
+----+------+------------+
| 1  | Will | null       |
| 2  | Jane | null       |
| 3  | Alex | 2          |
| 4  | Bill | null       |
| 5  | Zack | 1          |
| 6  | Mark | 2          |
+----+------+------------+
输出：
+------+
| name |
+------+
| Will |
| Jane |
| Bill |
| Zack |
+------+
```



# 二、解题思路\解题过程

就是一道简单的SQL题目，话不多说，代码如下所示：

```mysql
# Write your MySQL query statement below
select name from Customer where referee_id != 2 or referee_id is null;
```
