# 一、题目描述

## **问题概括**

给定一个长度为偶数 `n` 的整数数组 `nums`，你需要将数组中的元素两两配对，形成 `n/2` 个数对，每个元素恰好使用一次。每个数对的和定义为两个元素之和。你的目标是找到一种配对方案，使得所有数对和中的**最大值**尽可能小，并返回这个最小的最大值。

## **示例说明**

- **示例1**：`nums = [3,5,2,3]`，最优配对为 `(3,3)` 和 `(5,2)`，数对和分别为 6 和 7，最大值为 7。
- **示例2**：`nums = [3,5,4,2,4,6]`，最优配对为 `(3,5)`、`(4,4)`、`(6,2)`，数对和分别为 8、8、8，最大值为 8。

## **数据范围**

- `n` 是偶数，`2 ≤ n ≤ 10^5`
- `1 ≤ nums[i] ≤ 10^5`

# 三、代码演示

## Java代码

```java
class Solution {
    public int minPairSum(int[] nums) {
        Arrays.sort(nums);
        int n = nums.length, result = 0;
        for (int i = 0; i < n / 2; i++) {
            result = Math.max(result, nums[i] + nums[n - i - 1]);
        }
        return result;
    }
}
```



## Python代码

```Python
class Solution:
    def minPairSum(self, nums: List[int]) -> int:
        nums.sort()
        n, result = len(nums), 0
        for i in range(n // 2):
            result = max(result, nums[i] + nums[n - i - 1])
        return result
```
